Flexbox (flexible box model) to sposób wyświetlania elementów strony zaprojektowany dla potrzeb implementacji UI. Podstawą flexboxa jest to, że kontener może bezpośrednio wpływać na rozmiar i układ swoich dzieci. Najprostsza implementacja: 

.container {
	display: flex;	
}
.item {
	flex: 1;
}
